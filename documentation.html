Twitter Feed Documentation

This function calls out to twitter to get the last tweets for a user provided twitter handle
It has functions to create OAUTH connection with user provided values and create a string and dataframe of the tweets
Link to Tweepy Python Page

Sentiment over time Documentation

This function returns a plot of the tweets sentiment over time
This is accomplished with MatPlotLib and TextBlob sentement functions
Link to TextBlob Documentation

LDA Documentation

This function does topic Modeling on the tweets using the pyLDAvis
This is accomplished using functionality of the pyLDAvis topic visualization medels and takes a string input
Link to pyLDAvis

WordCloud Documentation

This function returns a wordcloud of the tweets
This is accomplished by using the Textblob Noun_phrases function
Link to WordCloud documentation
